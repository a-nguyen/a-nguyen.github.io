Run Javascript Code Within Sublime

What

  • A way to set up your Sublime Text editor to run javascript files and see their output within your editor window.
  • A REPL like experience from within Sublime Text.

Why

  • Context switching between Sublime and the console or a REPL slows you down a lot
  • Debugging from within the console often requires you to retype the same console.logs after every refresh
  • All the advantages of Sublime Text along with the interactivity of the Node REPL

How

  • In Sublime Text go to the toolbar and select Tools > Build System > New Build System
  • Install a version of Node.js (preferably through NVM)
  • Type the following to determine where your node.js binary command is located
  • This should return the |PathToNode|
  • Replace the template JSON with the following JSON with |PathToNode| replaced with the correct path
  • Save but DO NOT CLOSE the build file.
  • As you save rename it to something like Node.sublime-build
  • Open and save a test javascript file test.js
  • Put in some javascript code along with some “Hello World” style console.logs
  • Select Menu Item: Tools > Build System > Node
  • Press Command B on Mac or Ctrl B on Windows
    • This saves the js file
    • It runs the js file in node
    • It outputs the node output to a window frame at the bottom of sublime
    • The window can be resized and you can scroll up and down the output window
    • If everything worked correctly you should see your console.logs or an error if your js is not valid
  • You can now close your saved Node.sublime-build file
